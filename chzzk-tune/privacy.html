치지직튠 개인정보처리방침

최종 업데이트일: 2025년 4월 20일

1. 수집하는 정보

치지직튠은 사용자의 개인 정보를 수집하거나 저장하지 않습니다. 모든 기능은 사용자의 브라우저 내에서 로컬로 실행되며, 외부 서버와의 통신이나 데이터 전송이 발생하지 않습니다.

2. 저장되는 데이터

사용자가 설정한 인터페이스 옵션(예: 추천 라이브 숨기기, 좌측 패널 가리기 등)은 브라우저의 로컬 저장소(chrome.storage.local)에 저장되며, 외부로 전송되지 않습니다.

3. 제3자와의 데이터 공유

치지직튠은 어떠한 개인 정보도 제3자와 공유하지 않습니다.

4. 보안

치지직튠은 데이터를 외부로 전송하지 않으며, 사용자의 브라우저 내에서만 작동하므로 별도의 보안 조치가 필요하지 않습니다.

5. 아동의 개인정보

치지직튠은 아동의 개인정보를 수집하거나 저장하지 않습니다.

6. 정책 변경

본 개인정보처리방침은 변경될 수 있으며, 변경 시 본 페이지를 통해 공지합니다.
